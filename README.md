# 🧠 Real Log Analyzer Tool (Mini SIEM)

## 📌 Description
This tool analyzes real log files to detect suspicious login activity such as brute force attacks and repeated failed login attempts.

It simulates a basic Security Information and Event Management (SIEM) system used in cybersecurity operations.

## 🎯 Features
- Reads real log files (.txt)
- Detects failed login attempts
- Identifies suspicious users
- Detects suspicious IP addresses
- Generates security report


Make sure `logs.txt` is in the same folder.
